## PLATFORM SUMMARY

Most Tennessee cooperatives use one of these outage map platforms:

1. **OutageMap.coop (Kubra-based)** - Used by 10+ cooperatives
   - Standard URL format: `https://[utility].outagemap.coop/`
   - Modern, interactive maps
   - Real-time updates

2. **Custom/Legacy Systems** - Used by some utilities
   - Various URL formats
   - May require phone-based reporting

3. **No Public Map** - Caney Fork (at least 1 confirmed)
   - Phone-based outage reporting only
